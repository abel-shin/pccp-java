# 문자열

## 문자열 탐색

- 문자열(String)은 수정이 불가능하기 떄문에, 내용을 탐색하는 것이 주로 출제
- 문자열은 실무에서도 빈번하게 다루게 되므로, 꼭 잘 익혀두어야 한다!
- 문자열의 메소드를 잘 조합하여 필요한 기능을 구현하는 것이 핵심
- 다루는 문자열이 점점 길어질 경우, **StringBuilder**를 사용하여 효율적으로 구현
